# 常用Api

### 操作数组

```shell 
splice()      拼接
slice()	      切
replace()	  替换
split()		  分割，分离
concat()	  合并
join()		  添加
shift()       用于删除数组的第一项，并返回删除的值，每次只能删除一个，会改变原数组。
unshift()     用于在数组的第一个位置插入一个或多个元素，参数靠前的后插入，会改变原数组。
pop()         数组末尾的项删除，并且返回删除的值，每次只删除一个，会改变原数组。
push()		  用于在数组的最后插入一个或多个值，并返回数组的长度，会改变原数组。
arr.split('').reverse('').join()   翻转
indexOf() 	  返回要查找的项的索引，没有匹配的则返回-1
```
